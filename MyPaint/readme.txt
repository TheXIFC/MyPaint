Используется библиотека SFML 2.0
Используется шрифт Arail

Интерфейс программы:
4 поля поля для ввода чисел, 3 для задания цвета в RGB и 1 для задания ширины линий
поле для отображения заданого цвета
иконки для выбора инструмента рисования

При нажатии Delete поле очищается
При нажатии CTRL+Z удаляется последний нарисованый елемент
